# CUMT_NET

快速连接 CUMT 一体化网络

## 简介：

> CUMT_NET 是一个用于快速连接中国矿业大学一体化网络的脚本程序，省去每次登录繁琐的操作，具有可视化图形界面，仅需简单配置谷歌浏览器驱动程序即可立即使用
